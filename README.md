# 🌿 NatureVerse – Beautiful Nature Gallery Website

A modern and responsive **nature-themed gallery website** built using HTML and CSS.
This project showcases stunning nature visuals with smooth animations, clean UI, and an immersive user experience.

---

## 🚀 Features

* 🌄 Beautiful Nature Gallery Layout
* 🎨 Modern and Clean User Interface
* 🌌 Animated Background (Particles Effect)
* 📱 Fully Responsive Design
* 🧭 Navigation Bar with Smooth Sections
* 🖼️ High-Quality **4K HD Image Support**
* ⚡ Fast and Lightweight

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Google Fonts
* Font Awesome Icons
